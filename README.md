## Overview

I build full-stack web systems with a focus on backend architecture — the part of the product users never see, but always feel. My work sits primarily in the Laravel ecosystem, extending into React through Inertia.js and into mobile through Flutter.

I approach engineering as a long-term discipline: code should still make sense to whoever reads it a year from now, including myself.

<blockquote>
Systems are judged not by how fast they were built, but by how long they stay easy to change.
</blockquote>
